# HAR2JMX
## Version 0.0.14.1
### Date: 2022/10/04

-------------------------------

#### Description 
A free tool (based on .NET Core) to convert an HTTP Archive (*.har) file of recorded requests into an Apache JMeter Test Plan (*.jmx) file
